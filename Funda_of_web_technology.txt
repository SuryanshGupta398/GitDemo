Protocol:
    >> Set of rules/instructions
        -> http: Hyper Text Transfer Protocol
        -> https: Hyper Text Transfer Protocol Secure
        -> tcp: Transmission Control Protocol (Handshake signals)(Much More secure and fast)(connection btw sender and receiver for persistent amount of time)
        -> udp: User Datagram Protocol (Less secure)
        -> smtp: Simple Mail Transfer Protocol
        -> ftp: File Transfer Protocol
        -> ip: Internet Protocol(IPv4 and IPv6)
