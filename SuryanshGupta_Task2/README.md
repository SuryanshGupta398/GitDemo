In this task i have built a basic HTML page for Laundary Wallah / Laundary Mart. I use the header, main, footer sections to divide the webpage properly. 

This task covers the concept of various HTML tags, elements, and attributes like h1, table, form, img, etc.

First I created the header section using h1 tag to display the welcome message. 

Then I started the main section which first conatins a brief paragraph about the Laundary Mart. After that, I added one laundary related image (using the concept of img tag). 

For sub topics I used the h2 tag.

Next comes the Services section, which describes the services offered by laundary mart. I used an unordered list (using the concept of ul and li tags) to display the services.

Then, I created the Price List section, which shows the charges for different laundary services. To display this, I used a table (using the concept of table tag with the border attribute and th, tr, and td tags).

For Book Now section, i used a form(using the concept of form tag with its attributes and tags like input, label). For Name input type as text, for Email input type as email, for Phone input type as tel; and all fields were marked as required. And added a Submit button.

At last, I created the footer section to display contact information for Laundary Mart (I added the email address to contact the laundary and when email got clicked it directly open the email compose page it uses the concept of anchor tag).

Through this task, I understood the basic concepts of HTML tags, attributes, and elements and successfully applied them in building a simple webpage.

To run this project, open it in VS Code and install live server extension and on bottom right corner click Go live and then on browser it will open the page displaying all sections of the Laundary Mart webpage. 

At the bottom, there is a form. When the form is filled and submitted, the page only refreshes because no form action or JavaScript functionality has been added yet. Form processing works properly when backend integration or JavaScript is used.
